# Скрипты для исправления электронного дневника

Набор инструментов для автоматизации правок в базе данных. Позволяет массово исправлять оценки, удалять замечания учителей и добавлять похвальные отзывы.

## Как установить

1. Скачайте файл `scripts.py` из этого репозитория.
2. Положите его в корневую директорию проекта (туда, где лежит файл `manage.py`).

## Функционал и запуск

Все команды выполняются через [Django Shell](https://djangoproject.com). Чтобы войти в него, запустите в терминале:

```bash
python manage.py shell
```

Затем импортируйте функции:

```python
from scripts import fix_marks, remove_chastisements, create_commendation
```

### 1. Исправление оценок
Меняет все двойки и тройки на пятёрки для конкретного ученика.

```python
from datacenter.models import Schoolkid
child = Schoolkid.objects.get(full_name__icontains='Фролов Иван')

fix_marks(child)
```

### 2. Удаление замечаний
Стирает все плохие записи в дневнике ученика.

```python
remove_chastisements(child)
```

### 3. Добавление похвалы
Находит последний урок по указанному предмету и добавляет случайную похвалу от учителя.
Функция защищена: если ученик не найден или их несколько (например, "Степан"), скрипт сообщит об ошибке.

```python
create_commendation('Фролов Иван', 'Математика')
```
